# Line Work + Shading (DKmrB)

## Flow
1. Outline (construction lines or basic shapes -> silhouette).
2. Fill with solid base colors.
3. Shade: base -> shadow -> highlight.

## Clean lines
- Turn ON "pixel perfect" (Aseprite: `Edit > Preferences > Editor > Pixel Perfect` / brush snapping) to avoid DOUBLES — extra pixels that make right-angle spikes and look sharp/messy.
- Doubles are OK only as an intentional retro/bold style choice; keep consistent.
- Smooth a curve: start from both ends, meet in the middle; count pixels and repeat the SAME segment length on each side (symmetric). Gradually build up/down pixel counts.

## Light + perspective
- Pick ONE light direction; keep it constant across the whole game.
- 3/4 / oblique view: light from TOP of screen, cast forward+down. Tops of objects most lit.
- Tables/chairs: show the SAME amount of "top" for objects at similar heights.

## Shading techniques
- **3-shade ramps** (base/shadow/highlight) — the default.
- **Analogous shading** — highlight/shadow are neighboring hues, not lighter/darker base.
- **Dithering** — 2-color checker (or any pattern) to fake a 3rd color / show texture / break a big shadow. Originated from old consoles' limited palettes; now a style choice.
- **Anti-aliasing** — place an in-between color only at the CORNERS to soften. Makes art less "pixel-y"; optional, preference.

## Outlines as a focus tool (not just black)
- Recolor outline to a darker shade of the object (plants = dark green, wood = dark brown).
- Or bright accent (Omori = purple) for dreamlike identity.
- Inconsistent outline = guide the eye: Celeste outlines ONLY the player; everything else uses shading. AK outlines characters+NPCs+items, dims the world.
- Thick vs no-outline is an identity choice (Hyper Light Drifter = none; needs super-readable shapes + contrast vs background).
- You can shade the outline itself by light side = lighter, shadow side = darker.

## Habits
- Zoom out constantly (Aseprite F7 preview window) — judge at FINAL on-screen size.
- Use references (Pinterest > raw image search for less AI sludge).
- Bold + simple > complex. Max 3 shades/material.
